Реализация задачи:

- В качестве эммулятора используется PyQt5
- Для создания окон приложения используется Qt Designer в формате .ui:
  Установка скриннов в виде обоев окна,
  Установка тригерров в подсвеченные зоны.
- Компилирование .ui в .ру при помощи утилиты pyuic из командной строки
- Для реализации функционала приложения используется ООП Python с библиотеками(PyQt5, sys, os).
